## ğŸ“Œ Getting Started  

### 1. Use this template  

Click the **"Use this template"** button on GitHub to create your own repository from this starter.  

Or clone directly:

```bash
git clone https://github.com/stackovermaw/portfolio.git
cd portfolio
```

---

### 2. Install dependencies

Make sure you have **Node.js (>=18)** and **npm** or **pnpm** installed.  

```bash
npm install
```

---

### 3. Modify site content

Edit **`site.config.ts`** to customize your:  

- Name & about you
- Social links  
- Projects  
- Skills

Your **tech stack** can be edited at **`TechStackSection.tsx`**

---

### 4. Update metadata

Inside **`layout.tsx`**, update the **SEO metadata**:

- Page title  
- Description  
- Open Graph / Twitter cards  
- and etc.

This helps your portfolio look great when shared online.
